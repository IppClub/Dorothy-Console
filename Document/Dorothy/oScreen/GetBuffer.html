oScreen类的成员函数，获得当前图形设备的绘图缓冲区。

CHAR_INFO* GetBuffer();

参数 返回的是当前绘图缓冲区的首地址

——绘图缓冲区是一个被当作二维数组的一维数组，它的宽和高可以通过oScreen::GetWidth()和oScreen::GetHeight()获得然后你可以这样像二维数组一样遍历它

int iHeight = oScreen::GetSingleton()->GetHeight();

int iWidth = oScreen::GetSingleton()->GetWidth();

CHAR_INFO* pkBuffer = oScreen::GetSingleton()->GetBuffer();

for (int y = 0;y < iHeight;y++)

{

for (int x = 0;x < iWidth;x++)

{

pkBuffer[y*iWidth + x] = kCharInfo;

}

}
