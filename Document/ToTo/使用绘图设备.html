函数说明

void oSetScreenView(int X, int Y);

说明：设置可见区域在游戏世界中的位置，(X, Y)为游戏世界中的坐标

int oGetScreenViewX();

说明：获得可见区域左上角在游戏世界中的横坐标

int oGetScreenViewY();

说明：获得可见区域左上角在游戏世界中的纵坐标

void oBeginPaint(WORD fill = BACK_BLACK);

说明：开始绘制图形并按需要清理屏幕，fill为屏幕清理成的颜色，当为_NONE的时候不作清理，可用的颜色参数见后面的部分。在调用这个函数后面才应该进行图形的绘制

void oEndPaint();

说明：结束绘制图形，并把绘制结果显示在屏幕上，所有绘制图形的操作应该在这个函数前进行

oImageID oLoadFromBMP( const char* filename );

说明：载入一个24位的bmp图像，只有Windows绘图程序默认颜色的前16种会被载入，其它颜色会被当成透明部分。Filename为图片文件名，如果载入成功返回值为载入图片的ID，如果载入失败则返回值为-1。一定要小心保存好图片ID在后来使用

void oDrawImage( oImageID imageID, int X, int Y );

说明：绘制一个图片。imageID为图片ID，(X, Y)为图片左上角在游戏世界的坐标。所有的Draw函数只能在oBeginPaint和oEndPaint中间调用，否则不能正常绘制

void oDrawPoint(const oPoint* pointList, oUInt number, WORD , char ch = ' ');

说明：绘制点。pointList为一个点数组的开始指针，number为这个数组中点的数量，是点的颜色，ch为点上的字符。可用的颜色参数见后面的部分

void oDrawLine(const oPoint& start, const oPoint& end, WORD , char ch = ' ');

说明：绘制线。start为线在游戏世界的开始坐标，end为结束坐标，为线的颜色，ch为线上的字符。可用的颜色参数见后面的部分

void oDrawCircle(const oPoint& center, int radius, WORD , char ch = ' ');

说明：绘制圆。center为圆在游戏世界的中心坐标

void oDrawRectangle(const RECT& rect, WORD , char ch = ' ');

说明：绘制矩形。rect为矩形的区域，rect里的left，top为矩形在游戏世界的左上角坐标，right，bottom为右下角坐标，为颜色，ch为字符

可取的颜色参数

_NONE//透明色

FORE_BLACK

FORE_BLUE

FORE_GREEN

FORE_CYAN

FORE_RED

FORE_MAGENTA

FORE_YELLOW

FORE_WHITE

BACK_BLACK

BACK_BLUE

BACK_GREEN

BACK_CYAN

BACK_RED

BACK_MAGENTA

BACK_YELLOW

BACK_WHITE

FORE_INTENSITY//前景色增亮

BACK_INTENSITY//后景色增亮

前景色可以任意和FORE_INTENSITY组合

后景色可以任意和BACK_INTENSITY组合

前景色和后景色可以任意组合

例如颜色参数可以是这个样子

( BACK_GREEN )

( FORE_GREEN | FORE_INTENSITY )

( FORE_GREEN | FORE_INTENSITY | BACK_MAGENTA  | BACK_INTENSITY )

前后景色的意思见Dorothy中的Extra文档
