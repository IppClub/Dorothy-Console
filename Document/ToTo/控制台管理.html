函数说明

void oSetWindowTitle(char* title);

说明：设置控制台窗口的标题，title可以填入一个字符串

void oSetWindowSize(int width, int height);

说明：设置控制台宽高几各个格子，width为宽，height为高

void oMoveWindow(int ScrX,int ScrY);

说明：移动控制台窗口，ScrX和ScrY分别为要设置的控制台左上角在桌面上的坐标

void oSetFont(oUInt iFONT_WxH);

说明：设置控制台格子的分辨率，可以设置的值有FONT_3x5、FONT_5x8、FONT_6x12、FONT_8x16、FONT_8x18、FONT_10x20等

void oSetCursorPos(int x, int y);

说明：设置控制台光标的位置，以控制台屏幕左上角为(0,0)

void oSetCursorWidth(int percentWidth);

说明：设置光标的宽度，0看不到光标，100光标占满整个格子，只能取0~100的值

void oShowCursor(bool visible);

说明：设置是否显示光标，true显示，false不显示
