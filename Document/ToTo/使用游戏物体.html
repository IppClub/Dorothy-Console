函数说明

oObjectID oCreateObject(oImageID imageID, int X, int Y, int Z);

说明：创建一个物体。imageID为物体对应的图片ID，不能为不存在的图片ID值，(X, Y)为物体左上角在游戏世界的坐标，Z为物体的绘图顺序，Z值大的物体会遮挡Z值小的物体。创建出的物体大小和图片宽高相等。如果创建成功则返回物体的ID，失败返回ID_NONE这个数值

void oSetObjVisible(oObjectID objID, bool bVisible);

说明：设置物体是否可见。objID为物体的ID，bVisible为可见的标志，true可见，false不可见

void oSetObjDetective(oObjectID objID, bool bDetective);

说明：设置物体是否接受碰撞检测。objID为物体的ID，bDetective=true进行检测，false不进行检测

int oGetObjX(oObjectID objID);

说明：获得物体在游戏世界的横坐标。objID为物体的ID，返回横坐标。

int oGetObjY(oObjectID objID);

说明：获得物体在游戏世界的纵坐标。objID为物体的ID，返回纵坐标。

int oGetObjWidth(oObjectID objID);

说明：获得物体的宽。objID为物体的ID，返回宽几个格子。

int oGetObjHeight(oObjectID objID);

说明：获得物体的高。objID为物体的ID，返回高几个格子。

bool oIsObjVisible(oObjectID objID);

说明：物体是否可见。true可见，false不可见

bool oIsObjDetective(oObjectID objID);

说明：物体是否接受碰撞检测。True接受，false不接受

void oSetObjPosition(oObjectID objID, int X, int Y);

说明：设置物体的位置，objID为物体的ID，(X, Y)为物体左上角在游戏世界的坐标

void oSetObjZIndex(oObjectID objID, int Z);

说明：设置物体的Z序列值，objID为物体的ID，为物体的Z值，Z值大的物体会遮挡Z值小的物体

void oDrawObject(oObjectID objID);

说明：绘制物体。objID为物体的ID，这个函数只能在oBeginPaint()和oEndPaint()中间被调用

void oDrawAllObject();

说明：绘制所有存在的物体。这个函数只能在oBeginPaint()和oEndPaint()中间被调用

oObjectID oGetCollision(oObjectID objID, bool checkImage = false);

说明：检测物体的碰撞，objID为进行碰撞的物体，checkImage指定是否检查图片透明区域，如果为true则物体图片的透明区域重叠时不算碰撞。为false时只要物体的矩形区域相交就算发生碰撞。发生碰撞时返回被碰物体的ID，没有碰撞返回ID_NONE这个数值
