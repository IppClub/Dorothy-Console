使用输入设备

函数列表

void oUpdateInputState();

bool oIsRightButtonDown();

函数说明

void oUpdateInputState();

说明：更新并获取当前的用户输入状态，调用后可以通过其它函数获取当前的用户输入信息

bool oIsKeyDown(oUInt8 whichKey);

说明：获取用户是否按下某个按键，可取的参数见后面

bool oIsLeftButtonDown();

说明：获取用户是否按下鼠标左键

bool oIsRightButtonDown();

说明：获取用户是否按下鼠标右键

int oGetCursorX();

说明：获取当前鼠标在控制台上的横向第几个格子

int oGetCursorY();

说明：获取当前鼠标在控制台上的纵向第几个格子

键盘按键参数可取的值有

VK_0 ~ VK_9

VK_A ~ VK_Z

VK_F1 ~ VK_F12

VK_RETURN

VK_SHIFT

VK_CONTROL

VK_ESCAPE

VK_SPACE

VK_LEFT

VK_UP

VK_RIGHT

VK_DOWN
