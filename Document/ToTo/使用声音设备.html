函数说明

oSoundID oLoadFromWAV( const char* filename );

说明：载入一个.wav格式的声音，filename为声音文件名，如果载入成功则返回载入声音的ID，如果载入失败返回-1

void oPlaySound( oSoundID soundID, bool loop);

说明：播放指定ID的声音，soundID为声音的ID，loop为是否循环播放的标志，true循环，false不循环

void oStopSound( oSoundID soundID );

说明：停止指定ID的声音的播放，soundID为声音的ID

void oPlayMusic( const char* mp3file );

说明：播放MP3音乐，mp3file为要播放的文件名

void oStopMusic();

说明：停止正在播放的音乐

bool oIsMusicEnd();

说明：获取当前的音乐播放状态，如果返回值为true表示播放停止，false表示正在播放
